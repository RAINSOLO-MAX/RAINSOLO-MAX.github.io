一刀两半，一查到底：二分与差分的艺术

本文介绍两个经典的算法技巧：二分查找和二维差分，它们分别用于高效查询和矩阵区域修改。

一、二分查找

问题描述

给定一个升序排列的数组，对于每个查询值 x，找出 x 在数组中第一次和最后一次出现的位置。如果不存在，输出 -1 -1。

核心思想

二分查找的核心在于：每次将搜索区间减半，直到找到目标或区间长度为1。

要找到元素的起始和结束位置，需要进行两次二分查找：

  1. 找左边界：找第一个 >= x 的位置
  2. 找右边界：找最后一个 <= x 的位置

关键细节

细节 说明
mid = (l + r) >> 1 找左边界时，mid 向下取整
mid = (l + r + 1) >> 1 找右边界时，mid 向上取整，避免死循环
区间更新 左边界用 r = mid，右边界用 l = mid

时间复杂度：每次查询 O(log n)

核心思想

三、总结

算法 适用场景 时间复杂度
二分查找 有序数组中查找元素 O(log n)
